# N.C.R AI Academy — Web App espace membre

## Objectif
Cette web app transforme la formation IA N.C.R en portail membre premium :
- connexion stagiaire / formateur ;
- tableau de bord ;
- modules de formation ;
- bibliothèque de prompts filtrable ;
- diagnostic IA avec score ;
- espace documents ;
- certification avec QCM ;
- plan d’intégration IA 90 jours ;
- mode formateur pour suivre les stagiaires.

## Accès de démonstration
- Stagiaire : `stagiaire` / `stagiaire123`
- Formateur : `admin` / `admin123`

## Installation simple
1. Dézipper le dossier.
2. Ouvrir `index.html` dans un navigateur.
3. Pour héberger : envoyer les fichiers sur GitHub Pages, Netlify ou ton hébergement.

## Important
Cette version est une web app front-end premium.
Les comptes sont simulés côté navigateur et les données sont stockées en localStorage.

Pour une version production avec vrais comptes :
- Supabase ou Firebase pour l’authentification ;
- base de données pour progression / documents / scores ;
- stockage sécurisé pour PDF et certificats ;
- règles RGPD et politique de confidentialité dédiées.
