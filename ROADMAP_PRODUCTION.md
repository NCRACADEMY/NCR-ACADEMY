# Roadmap production — N.C.R AI Academy

## Version 1 livrée
Web app front-end premium, responsive, utilisable comme prototype commercial ou démonstrateur client.

## Version 2 connectée
- Authentification réelle stagiaire / formateur.
- Base de données apprenants.
- Progression sauvegardée en ligne.
- Documents protégés.
- Certificats générés automatiquement.
- Espace admin complet.

## Stack conseillée
- Front : React / Next.js.
- Auth & DB : Supabase.
- Storage : Supabase Storage.
- Hébergement : Vercel ou Netlify.
- Emails : Brevo ou Resend.
